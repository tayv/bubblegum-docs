# Calendar

Calendar is a primitive to be used by the [Date Pick component](../formDatePick). Displays an interactive calendar.
<br/>

## Current Status

- Built with date-fns, Radix UI Select, and Tailwind CSS
- Intended to be used within the [Date Pick component](../formDatePick)

## How to use

<Steps>
 
### Install date-fns library
 
```js copy 
npm install date-fns
```
It's used for date formatting.

### Install Radix UI Select
 
```js copy 
npm install @radix-ui/react-select
```
Used to build the year dropdown.

### Create component

Copy the code from the sandbox above and paste it into a new file (e.g. `Calendar.tsx`)
 
### Use it with Date Pick

Import it into [Date Pick](/formDatePick)

</Steps>
