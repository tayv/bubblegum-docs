# Radio Group

The Radio Group's used to select a single option from a list. It's built around the [Radix UI Radio Group](https://radix-ui.com/primitives/docs/components/radio-group) for better accessibility.
<br/>

<EmbedIframe url={"https://codesandbox.io/p/sandbox/bubblegumui-radiogroup-g0mh3d?embed=1"} />

## Current Status

- Built around Radix UI Radio Group primitive
- Basic styling using Tailwind CSS
- Four variants set via `type` prop: `standard`, `horizontal`, `button`, `buttonLarge`
- Works as a controlled component when used within the [Field component](/formControl/formField)

## How to use

<Steps>

### Install Radix Radio Group primitive

```jsx copy
npm install @radix-ui/react-radio-group
```

### Create component

Copy the code from the sandbox above and paste it into a new file (e.g. `RadioGroup.tsx`)

### Make it controlled

Wrap with [Field component](../formControl/formFieldField) to add control via React Hook Form:

```jsx copy jsx filename="yourform.tsx"
<Field
  name="radioButton"
  defaultValue="mario"
  >
  <Field.GroupLabel>Group label text here:</Field.GroupLabel>
  <Field.Control>
    <RadioGroup
      options={[
        { value: "mario", label: "Mario", icon: <Wrench /> },
        { value: "luigi", label: "Luigi", icon: <Flashlight /> },
        { value: "wario", label: "Wario", icon: <Zap /> },
        { value: "waluigi", label: "Waluigi", icon: <Skull /> },
      ]}
      type="buttonLarge"
    />
  </Field.Control>
</Field>
```

### Setup `defaultValues` and `zodSchema` in `Form`

Details in [Form component](../formControl/form)

</Steps>
