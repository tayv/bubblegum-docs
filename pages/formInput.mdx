## How to use

<Steps>

### Create component

Copy the code from the sandbox above and paste it into a new file (e.g. `Input.tsx`)

### Make it controlled

Wrap with [Field component](../formControl/formField) to add control via React Hook Form:

```jsx copy jsx filename="yourform.tsx"
<Field
  name="example"
  defaultValue={true}
  >
  <Field.GroupLabel>Group label text here:</Field.GroupLabel>
  <Field.Control>
    <Input type="text" size="standard" />
  </Field.Control>
</Field>
```
